# h1 医疗领域中术语及关系抽取的研究与应用

# h2 随着医疗信息系统的发展，医疗信息的数据与日俱增。由于用户的认知能力差异以及自然语言的二异性，导致医疗信息检索存在一定的盲目性。
     本体（Ontology）原属于哲学范畴，后被引入计算机人工智能研究领域。本体是对概念的明确结构化描述。
     在某一特定领域中，本体是该领域内全部概念及概念间关系所构成的知识体系。
     由于本体具有良好的概念层次结构并且支持逻辑推理，近年来，本体成为计算机领域内的研究热点，在信息检索领域中得到广泛的关注。

# h2 研究的意义：
        构建本体的意义：领域本体已经被广泛应用于知识工程和人工智能等多个领域，对医疗领域的知识管理起着关键作用。
        随着医疗信息系统的发展，医疗信息的数据与日俱增。由于用户的认知能力差异以及自然语言的二异性，导致医疗信息检索存在一定的盲目性。
        而本体可以形式化地表达问题，提供规范化、统一的知识表达形式，为知识的共享和重用提供模型，所以在辅助诊疗系统中引入领域本体进行知识管理至为重要。

        自动化构建的意义：
        当今移动互联网浪潮的推动之下，更是加快了领域知识的传播和信息积累，海量数据下继续沿用人工的方法代价太大，无法及时捕捉领域的发展。与此同时，
        计算机技术的迅猛发展给人们带来了新的希望。伴随着计算机软硬件水平的快速提升，利用计算机强大、准确而不知疲倦的计算能力来实现领域概念的自动抽取这一想法和需求也就产生了。

#  h2 辅助诊疗：
    对象：各级医生
    目的：在医生诊疗全程提供实时动态辅助决策，提高工作效率，弥补疏漏和纠正错误，降低误诊漏诊率
    特点：精准推送、标准诊断、精准治疗、规范记录

# h2 本研究旨在对本体的理论、方法、技术进行研究和实践，探索基于本体的信息整合方法；
     建立基于本体的领域知识体系；构建基于本体的医疗领域的信息检索系统，为医疗信息的检索提供方法和工具的支持。

# h2 研究工作主要包括：
     1、系统地学习了对于本体的概念、分类、作用、表示语言、构建方法和构建工具等理论及
        应用技术。
     2、对本体构建的方法和技术进行调研，比较分析和现有的本体构建方法，采用...，以《妊娠合并
        糖尿病诊治指南》为构建本体的主要依据，并参考《中华人民共和国药典》构建一些领域的本体。
     3、研究循环神经网络，词向量、随机条件场（CRF）
     4、对完成的研究工作进行了总结，对下一步研究工作进行了展望。

# h2 背景：随着医疗信息系统的发展，各类医疗信息呈现爆炸式的增长。如何科学地管理和维护海量
     的医疗信息，如何有效地挖掘和利用海量信息中蕴含的知识，已经成为一项迫切而重要的研究课题。
     从某种意义上讲，领域概念是人类在认知过程中对于事物的抽象，是领域知识在文本中的一种表现形式，
     并在一定程度上反映出该领域的发展变化。领域概念通常在特定对的领域中使用较为频繁，而在其他领域
     则使用较少。
     领域概念的搜集和整理工作在以前是依靠人工完成，虽然准确率高但是费时费力。同时，往往
     要求人们具备一定的领域知识，跨领域的概念获取好需要领域专家的支持。随着各学科领域的发展，
     旧的概念可能飞起，新的概念快速涌现。当今移动互联网浪潮的推动之下，更是加快了领域知识
     的传播和信息积累，海量数据下继续沿用人工的方法代价太大，无法及时捕捉领域的发展。与此同时，
     计算机技术的迅猛发展给人们带来了新的希望。伴随着计算机软硬件水平的快速提升，利用计算机
     强大、准确而不知疲倦的计算能力来实现领域概念的自动抽取这一想法和需求也就产生了。

     按照是否由两个以上词语构词，可以将领域概念分为单词型和复合型两类。复合型领域概念经过
     分词程序处理后往往被拆分成多个单词，相对来说抽取难度更大。当然，单词新领域概念与复合型
     领域概念之间存在密切联系得，许多复合型领域概念自身就高喊了单词型领域概念的成分。因此，努力
     提高单词型领域概念的抽取效果对于复合型领域概念的抽取也是很有帮助的。
    
# h2 通过分析医疗指南，结合医生针对患者的诊疗活动可以概括为：通过检查手段（做什么检查）发现疾病的表现（什么症状）
     给出诊断结论（什么疾病），并基于诊断结论，给出治疗措施（如何治疗）。可以看出，医疗活动主要涉及4类重要信息：检查、症状、疾病和治疗
     
# h2 领域知识图谱多采用自顶向下的方法来构建本体。
     一方面，相对于开放域知识图谱，领域知识图谱涉及的概念和范围都是固定或者可控的；另一方面，对于领域知识图谱，我们要求其满足较高的精度。
     概念及概念间的关连，作为描述真实世界的知识模型。
